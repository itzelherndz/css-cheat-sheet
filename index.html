CSS Snippet Cheat Sheet

Click on a code snippet and it will automatically select the script, ready for you to copy!

Basic CSS Rule

selector { 
   property: value;
}

Pseudo Class

selector:pseudo-class {
   property: value;
}

Pseudo Element

selector::pseudo-element {
   property: value;
}

Flexbox Row

selector {
    display: flex;
    flex-flow: wrap;
}

Flexbox Column

selector {
 display: flex;
 flex-direction: column;
}

Linear Gradients

selector {
 background-image: linear-gradient( direction, color-stop1, color-stop2, ...);
}
